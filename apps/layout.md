# Layout

The standard format for apps is as follows:

```bash
/
├── p
│   └── ...
├── public
│   └── Apps
│       └── Todos
│           ├── Artifacts
│           │   ├── config.json
│           │   └── db.json
│           ├── Data
│           │   ├── app.js.min.tar.gz
│           │   ├── index.html
│           │   └── style.css
│           └── Source
│               ├── package.json
│               ├── README.md
│               └── src
│                   └── ...
└── private
    └── Apps
        └── Todos
            └── ... same structure of public ...
```
